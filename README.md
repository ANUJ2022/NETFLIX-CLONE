##  Setup Instructions

### 📄 1. Create `.env` File

Create a `.env` file in the root of the project and add the following:

PORT=5000
MONGO_URI=your_mongo_uri
NODE_ENV=development
JWT_SECRET=your_jwt_secret
TMDB_API_KEY=your_tmdb_api_key


### 📦 2. Install Dependencies
npm install

🛠 3. Build the App
npm run build


▶️ 4. Start the App
npm run start
